# Work Day Scheduler

<h2> Description: </h2>
This is a simple calendar application that allows users to save events for each hour of the day. 

## <h2> Functionality: </h2>
  * The current day is displayed at the top of the calendar and there are timeblocks from 9 am to 5 pm. </br>
  * If the hour has already passed, the timeblock will reflect gray. </br>
  * If it is the current hour, the timeblock will reflect red. </br>
  * If the hour is in the future, the timeblock will reflect green. </br>
  * Users will be able to save their events by clicking the save button. </br>
  * If they refresh the page, the saved event will stay on the page. </br>
